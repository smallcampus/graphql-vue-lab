# graphql-vue-lab

This is a research project on how to integrate varies framework including graphql.

the associated graphql backend project is https://github.com/smallcampus/graphql-koa-lab

To get started:
`npm run serve`

Feature:

+ Typescript
+ Apollo + Graphql
+ Vue auto routing
+ Composition API
